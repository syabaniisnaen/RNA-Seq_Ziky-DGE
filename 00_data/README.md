# Dataset Description

## Source
Public RNA-seq data from NCBI SRA (PRJNA485140)

## Biological System
Human monocyte-derived macrophages (hMDMs)

## Experimental Conditions
- Mock
- ZIKV-bystander
- ZIKV-infected

## Timepoints
12 h, 18 h, and 24 h post infection

## Sequencing
- Platform: Illumina
- Library layout: **Single-end RNA-seq**
